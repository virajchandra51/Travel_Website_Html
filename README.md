# Travel_Website

1.  Install dependencies:

        yarn

    or

        npm install

2.  Run Server:

        npm run dev

Your canvas piece should open up automatically at http://localhost:3000 

Tech Stack Used :

1. HTML5
2. CSS
3. Vanilla JavaScript
4. ViteJs
5. SwiperJS
6. GSAP

SwiperJS along with GSAP is love, love, love :)
